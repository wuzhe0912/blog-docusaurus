## 📚 Domain Overview

### 🌐 Browser & Network

브라우저 렌더링 원리, HTTP 프로토콜과 네트워크 전송 메커니즘을 탐구합니다.

### 📒 JavaScript Ecosystem

JavaScript와 TypeScript의 핵심 개념, 클로저, 프로토타입 체인 등 심층 분석.

### ⚛️ Frontend Frameworks

React와 Vue의 프레임워크 원리, 라이프사이클, 상태 관리와 모범 사례.

### 🎨 CSS & UI

스타일 처리, 레이아웃과 최신 CSS 기능.

### 🛡️ Web Security

CSRF/XSS 공격 원리부터 JWT/Session 인증 메커니즘의 완전한 구현 분석까지.

### 🛠️ Engineering & Tools

프론트엔드 엔지니어링 도구, 번들링 전략 및 Performance 최적화 고급 기법.

### 💼 Experience

면접 준비와 실전 경험 공유.

### ⌨️ Coding

Lodash 함수 직접 구현과 JavaScript 고급 연습.

### 📊 LeetCode

알고리즘 문제 풀이와 사고 노트.

### 🤖 AI

AI Prompts와 도구 활용 노트.
