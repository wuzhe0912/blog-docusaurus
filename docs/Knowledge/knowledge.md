## 關於知識庫 (Knowledge Base)

這個區塊是前端工程師的綜合練功房，我們打破了「理論」與「實戰」的界線，將所有學習資源依照 **技術領域** 進行整合。

在這裡，您可以同時找到：

- **深度概念 (Concepts)**：底層原理、語言特性與規範。
- **快問快答 (Quiz)**：面試常見題型與解析。
- **實戰專題 (Practice)**：特定場景的解決方案（如 Authentication, Performance）。

### 📚 領域導覽

#### 🌐 Browser & Network

探討瀏覽器渲染原理、HTTP 協定與網路傳輸機制。

#### 📒 JavaScript Ecosystem

涵蓋 JavaScript 與 TypeScript 的核心概念、閉包、原型鏈等深度解析。

#### ⚛️ Frontend Frameworks

React 與 Vue 的框架原理、生命週期、狀態管理與最佳實踐。

#### 🎨 CSS & UI

樣式處理、版面配置與現代 CSS 特性。

#### 🛡️ Web Security

從 CSRF/XSS 攻擊原理到 JWT/Session 認證機制的完整實作解析。

#### 🛠️ Engineering & Tools

前端工程化工具、打包策略以及效能優化 (Performance) 的進階技巧。

---

## 學習建議

1. **從原理出發**：先理解該領域的核心概念。
2. **自我檢測**：利用 Quiz 確認是否掌握關鍵細節。
3. **深入實戰**：查看 Engineering 或 Security 中的實作案例，將知識應用於專案。

> 當發現自己無法順暢闡述筆記的內容時，代表不夠淺白易懂。盡可能添加案例來強化記憶。
