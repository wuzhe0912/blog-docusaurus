Waktu benar-benar berlalu lebih cepat dari yang kita kira. Setengah dari tahun 2024 sudah berlalu, tapi saya tidak merasa kedisiplinan diri saya meningkat banyak. Tidak banyak yang bisa ditunjukkan, tapi saya akan mencatat beberapa hal.

<!--truncate-->

## Perusahaan

Di akhir tahun 2023, perusahaan tempat saya bekerja remote mengalami penurunan bisnis dan segera mem-PHK setengah departemen frontend. Sayangnya, saya termasuk di dalamnya, jadi saya harus kembali ke pasar dan mencari penghasilan berikutnya.

Saya akhirnya bekerja di sebuah perusahaan di Taipei. Ya, perusahaan itu tidak menawarkan kerja remote, jadi saya kembali ke rutinitas bolak-balik. Itu berarti lonjakan biaya yang besar -- saya harus membawa istri dan anak saya kembali ke Taipei untuk menyewa tempat tinggal dan menetap untuk bekerja.

Perjalanan satu arah 30 hingga 60 menit setiap hari, hampir dua jam pulang pergi. Ditambah biaya sewa dan pengasuhan anak di Taipei, kualitas hidup dibandingkan era kerja remote turun drastis. Saat saya tiba di rumah, hampir tidak ada energi tersisa untuk urusan sendiri -- sebagian besar waktu saya habis untuk mengurus anak.

## Sisi Teknis

Meskipun saya terus bilang belum banyak berkembang, kalau dilihat kembali, saya memang menyentuh beberapa hal selama enam bulan ini:

### Refaktor Sistem Admin Live Streaming

Saya secara mandiri mengambil alih dan merefaktor sistem manajemen backend live streaming, meningkatkan alur kerja dan performa loading halaman. Menambahkan fungsi preview real-time dan pengaturan rasio aspek foto streamer agar staf operasional di lapangan bisa bekerja lebih lancar. Pada akhirnya, saya meningkatkan kecepatan respons halaman sebesar 40%.

### Optimisasi Integrasi Voice Game

Merefaktor pendekatan integrasi voice frontend game, beralih dari impor file statis ke pengambilan nama file secara dinamis. Sebelumnya, menambahkan satu set file voice baru membutuhkan 1-2 hari waktu pengembangan; setelah refaktor, turun menjadi sekitar 2 jam -- peningkatan efisiensi sekitar 90%. Saya juga menyiapkan proses standar agar maintainer di masa depan bisa menanganinya sendiri.

### Penyiapan Component Library

Membantu merefaktor sistem backend game dengan membangun component library bersama dan dokumentasi pengembangan menggunakan `Storybook`. Ini pertama kalinya saya melewati proses penyiapan component library yang relatif lengkap.

## Personal Brand

Saya mendaftar kursus media dan suara, berencana memaksa diri untuk merekam video tutorial. Tapi sama seperti 2023, saya menunda lagi. Sudah dua tahun, tujuan ini masih terjebak di garis start. Prokrastinasi memang keras kepala.

## Kehidupan

Jujur saja, perasaan terbesar paruh tahun ini bukan terobosan teknis apa pun -- tapi kelelahan dari menjadi orang tua. Anak makin besar, butuh lebih banyak perhatian dan waktu, dan dengan energi yang terkuras oleh perjalanan bolak-balik, jam setelah kerja saya hampir sepenuhnya tersita.

Ke depannya, saya berharap bisa memprioritaskan kerja remote atau hybrid, dan pada akhirnya bebas dari [terikat pada kota tertentu](https://amaca.substack.com/p/how-to-become-a-millionaire-without).

## Catatan Tambahan

Di akhir Juli, saya meninggalkan perusahaan itu dan mendapatkan posisi fully remote dengan gaji lebih baik. Setengah tahun bolak-balik ditukar dengan hasil yang cukup memuaskan. Sekarang saya berharap bisa memanfaatkan fleksibilitas yang ditawarkan kerja remote dan benar-benar mulai mengerjakan hal-hal yang selama ini saya tunda.
