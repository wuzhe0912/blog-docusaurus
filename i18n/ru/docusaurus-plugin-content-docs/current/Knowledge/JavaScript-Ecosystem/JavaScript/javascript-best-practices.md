## 1. Почему не следует использовать `index` в качестве key в списках и итераторах?

> Почему не следует использовать `index` в качестве значения `key`?

В вопросах на собеседованиях по фреймворкам вроде `Vue` и `React` эта тема встречается очень часто:

- Что произойдёт, если нет key в `v-for`?
- Почему не следует использовать `index` в качестве key в списках и итераторах `React`?
