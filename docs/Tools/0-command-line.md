# Command Line

> Basic Command Line

操作 `command line` 的終端機，可以根據自己的需求安裝對應的工具，譬如使用 `windows` 的 `git-bash`，`cmder`，或是 `mac` 的 `iterm2`。

```bash
pwd           // 印出當前所在位置
ls            // 列出目前資料夾下所有檔案
ls -al        // 列出隱藏檔案
cd folder     // 切換資料夾
cd ..         // 回到上一層資料夾
clear         // 清空終端機畫面
```

```bash
mkdir folderName      // 建立一個資料夾
touch fileName        // 若檔案存在則改變最後修改時間，若檔案不存在，則建立檔案
```
