## 1. Perché non dovresti usare `index` come key nelle liste e negli iteratori?

> Perché non dovresti usare `index` come valore della `key`?

Nelle domande di colloquio per framework come `Vue` e `React`, questo argomento appare molto spesso:

- Cosa succede se non c'è una key in `v-for`?
- Perché non dovresti usare `index` come key nelle liste e negli iteratori di `React`?
