## 4. Suppose there is a number N and we require the sum of 1+...\_+N

```javascript
// use closure to save variable
function solve(num) {
  let sumNumber = 0;
  for (let i = 1; i <= num; i++) {
    sumNumber += i;
  }
  return sumNumber;
}

console.log(solve(10));
```
